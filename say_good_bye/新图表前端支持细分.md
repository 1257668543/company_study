# 极坐标图



1. 功能面板：增加柱图类型下极坐标Polar单选框，控制直角坐标与极坐标切换
2. echarts：必须配置：

```js
const options = {
  polar: {
    // 极坐标轴内外半径与圆心等配置
  },
  radiusAxis: {
  	// 径向轴配置，即直角坐标下Y轴	*
	},
  angleAxis: {
    // 角度轴配置，即直角坐标下X轴
  },
  series: {
    // 系列配置，即指标数据，需显示设置坐标系为极坐标系
    coordinateSystem: 'polar'
  }
}
```
